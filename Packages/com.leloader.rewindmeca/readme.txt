Pour utiliser la méca, le mieux c'est d'avoir les deux fenêtres Scene et Game:

1: Selectionner les GameObjects à tracker avec l'icone à gauche du nom
2: Déplacer un objet dans la scène

REWIND:
3:  Appuyer sur TAB pour rewind dans le temps (défini par le nombre de frame dans le RewindManager)

PAUSE:
3:  Appuyer sur SPACE pour mettre pause, dans le RewindManager, un slider apparait permettant de contrôler le temps
